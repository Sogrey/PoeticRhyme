# 书香诗韵 PoeticRhyme

专注儿童诗词启蒙的国学学习平台

## ✨ 特色功能

- 🎨 **水墨古风 UI** - 使用 shuimo-ui 水墨风 UI 框架
- 🌈 **多主题切换** - 清新/暖阳/静谧三种护眼主题
- 📜 **磨砂纸质感** - 仿真纸质纹理背景
- 📱 **响应式设计** - 适配宽屏与移动端
- 👶 **儿童友好** - 专为 6-12 岁儿童设计

## 🛠️ 技术栈

| 技术               | 说明                   |
| ------------------ | ---------------------- |
| Vue 3 + TypeScript | 渐进式 JavaScript 框架 |
| Vite               | 下一代前端构建工具     |
| shuimo-ui          | 水墨风 UI 组件库       |
| Vue Router         | 官方路由管理           |
| Pinia              | 状态管理               |

## 📁 项目结构

```
src/
├── assets/
│   ├── fonts/          # 字体文件
│   ├── imgs/           # 图片资源
│   └── main.css        # 全局样式
├── components/
│   └── layout/         # 布局组件
├── router/             # 路由配置
├── views/              # 页面视图
├── App.vue
└── main.ts
```

## 🚀 快速开始

```sh
# 安装依赖
npm install

# 开发服务器
npm run dev

# 类型检查
npm run typecheck

# 构建生产版本
npm run build

# 代码规范检查
npm run lint
```
