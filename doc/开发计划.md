# 书香诗韵 · 开发计划

## 一、项目概述

- **项目名称**：书香诗韵
- **项目类型**：国学启蒙学习平台（Vue 3 空项目）
- **核心功能**：古诗词、成语、经典名句的阅读、赏析、筛选与检索
- **目标用户**：小学生、儿童（6-12岁）、家长、教师
- **UI 框架**：shuimo-ui（水墨风 UI）
- **开发目录**：`e:\woekspace\study\PoeticRhyme`

## 三、数据来源

| 序号 | 名称                                                                         | 说明                                                                                                     |
| ---- | ---------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------- |
| 1    | [chinese-poetry](https://github.com/chinese-poetry/chinese-poetry)           | 最全中华古诗词数据库，唐宋两朝近一万四千古诗人，接近5.5万首唐诗加26万宋诗，两宋时期1564位词人，21050首词 |
| 2    | [chinese-poetry-zhCN](https://github.com/chinese-poetry/chinese-poetry-zhCN) | chinese-poetry 简体中文版本                                                                              |
| 3    | [chinese-poetry-mysql](https://github.com/Kooooooma/chinese-poetry-mysql)    | 基于 chinese-poetry 数据整理的 MySQL 格式数据，含唐宋诗、宋词、论语、诗经等                              |
| 4    | [Werneror/Poetry](https://github.com/Werneror/Poetry)                        | 收录先秦到现代共计85万余首古诗词，CSV 格式                                                               |
| 5    | [hxgdzyuyi/tang_poetry](https://github.com/hxgdzyuyi/tang_poetry)            | 全唐诗数据库，MySQL 格式，含作者表和古诗表                                                               |

### 数据量统计

| 来源   | 诗词数量    | 说明                          |
| ------ | ----------- | ----------------------------- |
| 唐诗   | ~49,195 首  | Werneror/Poetry               |
| 宋词   | ~21,050 首  | chinese-poetry                |
| 全诗词 | ~853,385 首 | Werneror/Poetry（先秦至现代） |
| 全唐诗 | ~43,030 首  | hxgdzyuyi/tang_poetry         |

### 推荐使用

- **chinese-poetry**：JSON 格式，结构清晰，适合前端直接使用
- **chinese-poetry-mysql**：MySQL 格式，便于复杂查询
- **Werneror/Poetry**：CSV 格式，字段简单（题目、朝代、作者、内容）

## 五、已完成功能

### 基础架构

```
src/
├── assets/
│   ├── fonts/          # 字体文件
│   ├── imgs/          # 图片资源
│   └── main.css       # 全局样式
├── components/
│   └── layout/
│       ├── AppHeader.vue     # 顶部导航（含主题切换）
│       ├── AppFooter.vue     # 底部版权
│       ├── AppLayout.vue     # 主布局
│       └── ThemeSwitcher.vue # 主题切换组件
├── router/
│   └── index.ts       # 路由配置
├── views/             # 页面视图
├── App.vue
└── main.ts
```

### 主题系统

- **清新（light）**：翠绿主色调，护眼舒适
- **暖阳（warm）**：暖橙色系，温馨舒适
- **静谧（cool）**：蓝色系，沉静专注

### 背景效果

- 磨砂纸纹理叠加
- 毛玻璃头部/底部导航
- 主题色平滑过渡动画
