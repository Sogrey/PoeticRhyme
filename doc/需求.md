# 书香诗韵 · 网站需求规划书

## 一、网站定位

面向**小学生、儿童、家长、教师**的国学启蒙学习平台，以**诗词、成语、名句**为核心，集阅读、赏析、检索、学习于一体，弘扬中华传统文化，轻松启蒙国学。

## 二、网站 Description（已确定）

书香诗韵，专注诗词歌赋、儿童诗词启蒙与小学古诗词学习。提供优质古诗赏析、趣味诵读、诗词动画与同步教材内容，让孩子轻松爱上诗词，感受中华传统文化之美。

# 三、功能总规划

1. **内容库**
   - 古诗词（唐诗、宋词、元曲、乐府、小学必背、启蒙诗词）
   - 成语（释义、出处、例句、故事）
   - 经典名句（诗词名句、古文名言、劝学励志类）

2. **多维度筛选体系**
   - 按类型：诗词 / 成语 / 名句
   - 按朝代：唐 / 宋 / 元 / 明 / 清 / 先秦 / 魏晋南北朝等
   - 按作者：李白、杜甫、苏轼、王维等
   - 按年级：小学一年级～六年级 / 启蒙
   - 按题材：写景、抒情、送别、爱国、田园、励志等
   - 按热度：最新 / 阅读量高 / 推荐

3. **核心页面结构**
   - 首页
   - 诗词频道页
   - 成语频道页
   - 名句频道页
   - 详情页（诗词详情 / 成语详情）
   - 搜索结果页

# 六、儿童友好设计规范（6-12岁低龄儿童）

## 6.1 界面设计原则

### 6.1.1 布局设计

- **宽屏设计**：桌面端采用宽松布局，诗词正文区域宽度建议 800-1200px，确保阅读舒适性
- **移动端适配**：窄屏设备采用单栏布局，内容自适配屏幕宽度，底部固定导航便于操作
- **视觉层次**：重要内容（如诗词标题、正文）置于视觉焦点区域，次要信息层次分明
- **留白充足**：避免信息过密，提供足够的呼吸空间，降低阅读疲劳

### 6.1.2 色彩规范

| 类别   | 色彩倾向           | 说明                           |
| ------ | ------------------ | ------------------------------ |
| 主色调 | 水墨绿、靛蓝、暖橙 | 取自传统文化元素，柔和不刺眼   |
| 背景色 | 米白、淡青、浅藕   | 护眼舒适，避免纯白背景         |
| 强调色 | 橙红、翠绿、明黄   | 用于重要按钮、标签，吸引注意力 |
| 文字色 | 深灰、墨黑         | 正文可读性强，不纯黑           |

### 6.1.3 字体规范

- **诗词正文**：使用项目本地字体（鼎猎宋体/I-Ngaan），字号不小于 18px，行高 1.8-2.0
- **标题字体**：使用珠海体，笔画清晰，便于儿童识别
- **辅助文字**：采用圆润无衬线字体，拼音字号建议 14-16px

### 6.1.4 图标与插图

- 图标采用圆润风格，线条粗细适中（2-3px），便于儿童理解
- 重要交互区域配有装饰性插图（如诗词配图、作者卡通形象）
- 避免抽象图标，优先使用具象化图形表达含义

---

## 6.2 交互设计规范

### 6.2.1 操作简化

- **点击区域**：可点击元素最小尺寸不低于 44x44px（触屏友好）
- **操作步骤**：核心功能不超过 3 步完成，如"首页→诗词详情→收藏"
- **手势支持**：支持点击、滑动（切换诗词/翻页），不依赖复杂手势
- **即时反馈**：按钮点击后有明确状态变化（颜色/动效），告知操作成功

### 6.2.2 导航设计

- **底部导航（移动端）**：固定显示 4 个核心入口（首页/诗词/成语/我的）
- **顶部导航（桌面端）**：横向导航，字号 16-18px，hover 状态明显
- **面包屑**：详情页显示当前位置，便于返回上级
- **返回引导**：移动端左上角固定返回按钮，桌面端支持浏览器后退

### 6.2.3 搜索优化

- **搜索框**：首屏大搜索框占主视觉区域，placeholder 提示示例
- **智能联想**：输入时实时显示匹配结果，支持模糊搜索
- **筛选器**：采用卡片式/标签式筛选，一键切换，无需复杂配置

---

## 6.3 内容呈现规范

### 6.3.1 诗词展示

- **拼音标注**：可选显示拼音，支持逐字点读
- **断句标识**：诗句间有明显分隔，利于朗读节奏
- **译文对照**：古文/译文双栏对照，或折叠式展开
- **朗读功能**：支持一键播放朗读音频（可选扩展）

### 6.3.2 赏析内容

- **儿童化表达**：注释与赏析采用简单易懂的语言，避免生僻词
- **趣味延伸**：可加入诗词相关小故事、背景动画
- **相关推荐**：同题材/同作者/同朝代诗词推荐，拓展阅读

### 6.3.3 成就激励

- **阅读计数**：记录阅读诗词数量，激励持续学习
- **收藏功能**：支持收藏喜欢的诗词，建立个人诗库
- **每日一诗**：推送功能，引导养成每日学习习惯

---

## 6.4 响应式断点设计

| 设备类型 | 屏幕宽度   | 布局策略                         |
| -------- | ---------- | -------------------------------- |
| 桌面端   | ≥1200px    | 多栏布局，诗词正文居中，两侧留白 |
| 平板端   | 768-1199px | 两栏或单栏自适应，导航保持       |
| 移动端   | <768px     | 单栏布局，底部固定导航，触屏优化 |

---

## 6.5 无障碍与可读性

- **色彩对比度**：文字与背景对比度不低于 4.5:1
- **字体缩放**：支持浏览器字体缩放，不破坏布局
- **焦点指示**：键盘导航时有明显焦点框
- **动画控制**：提供减少动画选项，尊重用户偏好
